Olá pessoal,

Esta semana durante o evento [Connect](https://www.microsoft.com/en-us/connectevent/) foi lançado o novo Visual Studio 2019 &#8211; Preview, sim ainda é uma versão inicial, mas já temos muitas novidades!!

A primeira delas é a nova pagina inicial, agora bem mais simples:

[<img class="alignnone wp-image-10265" src="http://carloscds.net/wp-content/uploads/2018/12/vs2019-home-300x205.png" alt="" width="828" height="566" srcset="http://carloscds.net/wp-content/uploads/2018/12/vs2019-home-300x205.png 300w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-home-768x525.png 768w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-home.png 1024w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-home-731x500.png 731w" sizes="(max-width: 828px) 100vw, 828px" />](http://carloscds.net/wp-content/uploads/2018/12/vs2019-home.png)

Somente 4 opções (lado direito) e a lista dos últimos projetos abertos.

Ao criar um novo projeto, temos uma nova tela::

[<img class="alignnone wp-image-10274" src="http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto-300x205.png" alt="" width="841" height="575" srcset="http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto-300x205.png 300w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto-768x525.png 768w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto.png 1024w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto-731x500.png 731w" sizes="(max-width: 841px) 100vw, 841px" />](http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto.png)

Eu, particularmente, gostei desta nova tela, onde você tem alguns filtros, como : linguagem, tipo de projeto, plataforma e também a busca por nome.

Neste exemplo vou criar um projeto Asp.Net Core:

[<img class="alignnone wp-image-10275" src="http://carloscds.net/wp-content/uploads/2018/12/vs2019-aspntecoreproject-300x205.png" alt="" width="836" height="571" srcset="http://carloscds.net/wp-content/uploads/2018/12/vs2019-aspntecoreproject-300x205.png 300w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-aspntecoreproject-768x525.png 768w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-aspntecoreproject.png 1024w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-aspntecoreproject-731x500.png 731w" sizes="(max-width: 836px) 100vw, 836px" />](http://carloscds.net/wp-content/uploads/2018/12/vs2019-aspntecoreproject.png)

E agora vamos preencher os dados do projeto:

[<img class="alignnone wp-image-10276" src="http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto2-300x205.png" alt="" width="833" height="569" srcset="http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto2-300x205.png 300w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto2-768x525.png 768w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto2.png 1024w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto2-731x500.png 731w" sizes="(max-width: 833px) 100vw, 833px" />](http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojeto2.png)

A tela seguinte não mudou muito relação ao Visual Studio 2017:

[<img class="alignnone wp-image-10277" src="http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojetotype-300x211.png" alt="" width="830" height="584" srcset="http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojetotype-300x211.png 300w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojetotype-768x540.png 768w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojetotype-711x500.png 711w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojetotype.png 786w" sizes="(max-width: 830px) 100vw, 830px" />](http://carloscds.net/wp-content/uploads/2018/12/vs2019-createprojetotype.png)

O interessante desta nova versão é que até este momento ainda não entramos, de fato, na IDE do Visual Studio!

Ao pressionar o OK para criar o projeto, finalmente temos a IDE:

[<img class="alignnone wp-image-10278" src="http://carloscds.net/wp-content/uploads/2018/12/vs2019-ide-300x105.png" alt="" width="846" height="296" srcset="http://carloscds.net/wp-content/uploads/2018/12/vs2019-ide-300x105.png 300w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-ide-768x268.png 768w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-ide-1024x358.png 1024w, http://carloscds.net/wp-content/uploads/2018/12/vs2019-ide-1025x358.png 1025w" sizes="(max-width: 846px) 100vw, 846px" />](http://carloscds.net/wp-content/uploads/2018/12/vs2019-ide.png)

Esta nova IDE está mais &#8220;clean&#8221;, com uma pequena mudança na cor e também está mais rápida!

Eu vou começar a utilizar a IDE como padrão para meus projetos, sim, aqui é &#8220;Dog Food&#8221; !!!

Se você ainda não instalou o Visual Studio 2019 Preview, este é o link: <https://visualstudio.microsoft.com/pt-br/vs/preview/>

Lembrando que você pode instalar juntamente com outras versões do Visual Studio!

Enjoy!
